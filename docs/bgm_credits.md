## 曲を追加・変更するときの手順

- **差し替え**: 新しい曲を該当の役割名で `godot/assets/audio/bgm/` に上書き → この表の曲名欄を更新
- **磨き画面の曲を増やす**: `game_d.mp3` として追加し、`main.gd` の
  `_show_game_screen()` にある `["a", "b", "c"]` に `"d"` を足す → この表に行を追加
- **画面の種類を増やす**: `game_audio.gd` 冒頭のID一覧コメントも更新すること

## 備考

- 再生システムは ogg / wav / mp3 対応（`game_audio.gd` の `_load_stream`）
- 全曲自動ループ。音量既定値は BGM 40%（`game_audio.gd` の `_volumes`）
- 元ファイル一式の聴き比べ用コピーは `/bgm_candidates/`（gitignore済・配布物に含めない）
