# 必要アセット一覧

最終更新: 2026-07-16（表情差分・FINISH演出・こすり判定 実装時点）

- **受け口実装済み** = 指定の場所にファイルを置く（またはパスを書く）だけでゲームに反映される
- **フック未実装** = 素材が来たらコード側の対応もセットで行う（もらえれば すぐ組み込む）
- 置き場所はすべて Godot プロジェクト内（`godot/` フォルダ配下）。画像は PNG、音は OGG Vorbis 推奨。
- ファイル名は小文字スネークケース。キャラ id は `general`（女将軍）/ `admiral`（エルフ提督）の2人。

## 1. キャラ表情立ち絵（最優先・受け口実装済み）

`godot/assets/chara/<キャラid>/<表情id>.png` に置くと自動で反映される（2人 × 11表情 = 22枚）。

- 表示領域: 幅440 × 高さ590px にアスペクト維持で収まる → **推奨キャンバス 880×1180px（2倍解像度・縦長）**
- **全表情で同一キャンバス・同一立ち位置**にすること（切り替え時にガタつくため）
- 背景透過PNG

| 表情id | 状況 | 表情（表情差分と音.xlsx 準拠） |
|---|---|---|
| `idle_a` | ブラシ無し・快感ゲージ低 | 怒り・軽蔑 |
| `idle_b` | ブラシ無し・ゲージ中 | 恥じらい・耐え |
| `idle_c` | ブラシ無し・ゲージ高 | 大快感耐え |
| `idle_d` | ブラシ無し・絶頂寸前 | 媚び |
| `touch_a` | ブラシ当て・痛み優勢 | 痛い |
| `touch_b` | ブラシ当て・ゲージ低 | 恥じらい・耐え |
| `touch_c` | ブラシ当て・ゲージ中 | 快感 |
| `touch_d` | ブラシ当て・ゲージ高 | 大快感 |
| `climax` | FINISH演出中（5秒） | 絶頂 |
| `despair` | 痛み限界（失敗演出2.5秒） | 絶望 |
| `exhausted` | FINISH後の余韻（4秒） | 憔悴 |

## 2. キャラ選択カードのポートレート（受け口実装済み）

2枚。`main.gd` のキャラ定義 `portrait` にパスを書く（例: `res://assets/chara/general/portrait.png`）。
表示はカード内にアスペクト維持で収まる。縦長推奨（立ち絵の流用でも可）。

## 3. オープニングCG（受け口実装済み）

各キャラ3ページ（ページ数は増減可、`opening_pages` の配列で管理）。
`main.gd` の `opening_pages[].image` にパスを書く。表示域は約1230×430の横長 → **16:9（1280×720）推奨**。
テキスト原稿も同じ場所に差し込む（現状は仮テキスト）。

## 4. 磨きターゲット画像（受け口実装済み）

2人 × 左右 = 4枚。`main.gd` のキャラ定義 `left/right` の `image` にパスを書く。
円形領域に伸縮表示される: 女将軍は直径220px、提督は直径196px → **512×512の透過PNG**で足りる。

## 5. ブラシ画像（フック未実装・現状は色付き円）

4枚: やわらか / かため / 羽 / 極細。当たり判定は半径14〜30pxの円なので、**128×128の透過PNG**目安。
素材が来たらコード側で `brush.gd` に画像表示を足す。

|---|---|---|
| 磨き画面の背景（牢獄） | 1280×720 | 立ち絵の背後。暗め推奨（HUDが白系のため） |

## 7. フォント（方針決定済み・組み込みはコード側）

| 用途 | フォント | 入手 | ライセンス |
|---|---|---|---|
| メインUI | しっぽり明朝B1 | Google Fonts / fontdasu.com | SIL OFL 1.1 |
| タイトル・FINISH・絶望演出 | 源界明朝 | flopdesign.com | SIL OFL 1.1 |

`godot/assets/fonts/` に ttf/otf を置く。**OFLのライセンステキストを同梱すること**。組み込み作業はコード側で行う。

## 8. 音（フック未実装・次の実装候補）

音の再生システム自体が未実装。素材と並行してコード側を作る。「表情差分と音.xlsx」の音側に対応。

| 分類 | 素材 | 備考 |
|---|---|---|
| BGM | タイトル用 1曲 | ループ |
| BGM | 磨き画面用 1曲 | ループ。暗め・静かめだと SE が立つ |
| SE | ブラシ接触ループ | 可能なら快感段階（弱/中/強）で3差分 |
| SE | 痛い反応 | touch_a（痛み優勢）用 |
| SE | 絶頂ジングル | FINISH演出（5秒）に合わせる |
| SE | 絶望スティング | 痛み限界の暗転（2.5秒）に合わせる |
| SE | 心音ループ | ゲージが絶頂寸前のとき |
| SE | UI操作音 | クリック / 決定 / 画面遷移 |
| ボイス | 未定 | 入れるなら表情と同じ11状態に合わせると組み込みが素直。有無・分量はユーザー判断 |

## 進め方メモ

- 効果が大きい順: **表情22枚 → 磨きターゲット4枚 → 背景 → オープニングCG → 音 → その他**
- 表情はまず `idle_a` / `touch_c` / `climax` の3枚だけ先行で入れて表示確認するのがおすすめ（残りは差分作業）
- 素材が1枚でも入ればプレースホルダ（表情名ラベル等）は自動で消える
