# 必要アセット一覧

最終更新: 2026-07-16（表情差分・FINISH演出・こすり判定 実装時点）

- **受け口実装済み** = 指定の場所にファイルを置く（またはパスを書く）だけでゲームに反映される
- **フック未実装** = 素材が来たらコード側の対応もセットで行う（もらえれば すぐ組み込む）
- 置き場所はすべて Godot プロジェクト内（`godot/` フォルダ配下）。画像は PNG、音は OGG Vorbis 推奨。
- ファイル名は小文字スネークケース。キャラ id は `general`（女将軍）/ `admiral`（エルフ提督）の2人。

## 1. キャラ表情立ち絵（最優先・受け口実装済み）

`godot/assets/chara/<キャラid>/<表情id>.png` に置くと自動で反映される（2人 × 11表情 = 22枚）。

- 表示領域: 幅440 × 高さ590px にアスペクト維持で収まる → **推奨キャンバス 880×1180px（2倍解像度・縦長）**
- **全表情で同一キャンバス・同一立ち位置**にすること（切り替え時にガタつくため）
- 背景透過PNG

| 表情id | 状況 | 表情（表情差分と音.xlsx 準拠） |
|---|---|---|
| `idle_a` | ブラシ無し・快感ゲージ低 | 怒り・軽蔑 |
| `idle_b` | ブラシ無し・ゲージ中 | 恥じらい・耐え |
| `idle_c` | ブラシ無し・ゲージ高 | 大快感耐え |
| `idle_d` | ブラシ無し・絶頂寸前 | 媚び |
| `touch_a` | ブラシ当て・痛み優勢 | 痛い |
| `touch_b` | ブラシ当て・ゲージ低 | 恥じらい・耐え |
| `touch_c` | ブラシ当て・ゲージ中 | 快感 |
| `touch_d` | ブラシ当て・ゲージ高 | 大快感 |
| `climax` | FINISH演出中（5秒） | 絶頂 |
| `despair` | 痛み限界（失敗演出2.5秒） | 絶望 |
| `exhausted` | FINISH後の余韻（4秒） | 憔悴 |

## 2. キャラ選択カードのポートレート（受け口実装済み）

初期・オープニング視聴後の計4枚を配置・接続済み。

- 女将軍: `godot/assets/chara/general/portrait.png` / `portrait_after_opening.png`
- エルフ提督: `godot/assets/chara/admiral/portrait.png` / `portrait_after_opening.png`

`characters.gd` のキャラ定義 `portrait` / `portrait_after_opening` から読み込み、
`opening_seen` に応じて切り替える。
表示はアスペクト比を維持し、画像全体がカード内に収まる最大サイズで中央配置する。
上部に選択案内、右下3分の1幅にプロフィールを重ねる。
カード全体をクリックすると確認ダイアログを表示し、承認後に遷移する。
各カードには、成長値とオープニング状態を初期化するデバッグ用リセットボタンを置く。
縦長推奨（立ち絵の流用でも可）。

## 3. オープニング演出（実装済み・専用CG不要）

各キャラ初回スタート時の3ページ構成（`characters.gd` の `opening_pages` で管理、増減可）:

1. `split`: 左半分にキャラ選択の初期立ち絵（`portrait`）、右半分に導入テキスト
2. `blackout`: 全画面真っ暗＋中央に「だが捕らえた。」
3. `split`: 左半分にOP済立ち絵（`portrait_after_opening`）、右半分に捕縛後テキスト

画面のどこをクリックしても次ページへ進む（暗転ページはクリックのみ）。
立ち絵はキャラ選択画面と共用するため、オープニング専用CGは不要。
ページの `portrait` キーにキャラ定義のキー名（`portrait` / `portrait_after_opening`）を書く。

## 4. 磨きターゲット画像 = 乳首部分のオーバーレイ（受け口実装済み）

2人 × 左右 = 4枚。裸の立ち絵の乳首部分だけを別画像にして重ね、そこが当たり判定になる。
`characters.gd` のキャラ定義 `left/right` の `image` にパスを書く。
表示は直径44〜48pxの円形領域に伸縮 → **128×128の透過PNG**で十分。
立ち絵と同じ拡大率で切り出すとなじむ。重ねる位置・当たり判定サイズはコード側で立ち絵に合わせて調整する。
（表情差分で立ち絵が切り替わっても、このオーバーレイは同じ位置に載り続ける想定。
表情ごとに乳首の位置や状態が変わる絵にする場合は事前に相談）

### 磨き画面のキャラ背景（配置・接続済み）

- 女将軍: `godot/assets/chara/general/game_background.png`
- エルフ提督: `godot/assets/chara/admiral/game_background.png`

推奨は**正方形（800×800〜1024×1024）の顔と胸のバストアップ**。磨き画面中央の640×720領域へ、
天地いっぱい・左右クロップ（cover表示）で収める。正方形素材なら左右が約11%ずつ切れるので、
顔と胸を中央に置き、左右端に重要な要素を置かないこと。縦長素材（960×1280等）も同じ規則で表示される
（その場合は上下が切れる）。
表情差分画像が存在する場合は表情差分を優先し、無い場合にこの固定背景を表示する。

## 進め方メモ

- 効果が大きい順: **表情22枚 → 磨きターゲット4枚 → 背景 → オープニングCG → 音 → その他**
- 表情はまず `idle_a` / `touch_c` / `climax` の3枚だけ先行で入れて表示確認するのがおすすめ（残りは差分作業）
- 素材が1枚でも入ればプレースホルダ（表情名ラベル等）は自動で消える
